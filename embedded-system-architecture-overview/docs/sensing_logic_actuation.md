# Sensing, Logic, and Actuation

## Main point

A strong embedded system separates:

- what is measured
- what is decided
- what is driven

## Good flow

1. sensor value is read
2. sensor value is interpreted
3. logic decides whether action is needed
4. actuator output is updated
5. status may be reported

## Why this matters

This makes the system easier to explain and easier to expand later.
