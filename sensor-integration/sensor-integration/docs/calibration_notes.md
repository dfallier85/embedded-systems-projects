# Calibration Notes

## Main point

Raw sensor values are hardware-specific and environment-specific.

That means calibration matters.

## Example idea

For a soil moisture style sensor, you might record:
- dry reading
- wet reading

Then map future values between those reference points.

## Why this matters

Without calibration:
- thresholds are arbitrary
- values are hard to explain
- behavior is less repeatable

## Good practice

Write down:
- dry reference reading
- wet reference reading
- how the percent or interpreted value is calculated
