# Embedded Systems Projects

## Overview
This repository contains embedded systems projects involving sensing, actuation, communication, and control using microcontrollers and supporting tools.

## Topics Covered
- Sensor integration
- Relay and actuator control
- MQTT-based embedded communication
- Embedded architecture planning
- Power, wiring, and testing workflows

## Repository Structure
- `embedded-system-architecture-overview/`
- `esp-mqtt-servo-control-clean/`
- `relay-actuator-control-system/`
- `sensor-integration/`

## Tools and Technologies
- ESP32
- ESP8266
- STM32
- Arduino-style development
- MQTT
- Node-RED

## Purpose
This repo is part of my robotics and automation portfolio and focuses on building reliable, hands-on embedded systems with real hardware.
